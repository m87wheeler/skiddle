# Welcome To My Skiddle App

Thanks for taking the time to take a look at the code for my Skiddle app!

Before proceeding, please note that `NEXT_PUBLIC_SKIDDLE_KEY` is a mandatory environment variable.

### Setup Development

```bash
cp .env.example .env
```

```bash
yarn
yarn dev
```

### Setup Production

```bash
cp .env.example .env
```

```bash
yarn build
yarn start
```
